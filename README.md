# KDR Revamped — Prototype Workspace

Minimal Next.js + TypeScript scaffold to start development for the KDR project.

Getting started (PowerShell):

```powershell
# install dependencies
npm install

# run development server
npm run dev
```

Project structure (initial):

- `pages/` — Next.js pages
- `components/` — shared React components
- `styles/` — global CSS

Next steps:
- Install Tailwind CSS and other UI libraries
- Add authentication provider (Clerk/NextAuth)
- Set up database (Postgres/Supabase)
